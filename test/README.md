# My First Project

## 📌 Описание
Этот проект создан для практики написания документации.  
Он демонстрирует основы работы с Python и форматирование текста в Markdown.  

## 🚀 Функции
- Вывод приветственного сообщения.  
- Подсчёт суммы двух чисел.  
- Проверка установленной версии Python.
